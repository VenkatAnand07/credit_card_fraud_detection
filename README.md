# 💳 Real-Time Credit Card Fraud Detection System

A Machine Learning-based system to detect fraudulent credit card transactions using **SMOTE + Ensemble Models (SVM, Random Forest, XGBoost)**, deployed with a **Flask API for real-time prediction**.

---

## 🚀 Key Highlights
- Detects fraudulent transactions with high accuracy
- Handles highly imbalanced dataset using **SMOTE**
- Uses **ensemble learning** for better performance
- Provides **real-time predictions via Flask API**
- Outputs:
  - Fraud probability
  - Risk level
  - Feature impact insights

---

## 📊 Model Performance
- **Accuracy:** ~99.9%
- **Precision (Fraud):** ~95%
- **Recall (Fraud):** ~85–90%
- **F1 Score:** ~0.90+
- **AUC-ROC:** ~0.97+

> ⚠️ Note: Accuracy can be misleading for imbalanced datasets. Recall and F1-score are more important in fraud detection.
